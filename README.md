Introduction
============

A Perl implementation of Irving's algorithm for the stable roommates problem.

* http://en.wikipedia.org/wiki/Stable_roommates_problem
* http://www.dcs.gla.ac.uk/~pat/jchoco/roommates/papers/Comp_sdarticle.pdf

Installation
============
    perl Makefile.PL
    make
    make test
    make install

Usage
=====

See bin/pair_up.pl
